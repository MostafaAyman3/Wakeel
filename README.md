# AERIE

Agentic ERP Intelligence Engine.

AERIE is a Docker Compose based, single-tenant MVP foundation for an agentic AI layer on top of Odoo ERP. It uses FastAPI, LangGraph, PostgreSQL, Redis, Next.js, TailwindCSS, GPT-4o / GPT-4o Mini, and LangSmith.

This repository is currently bootstrapped for Sprint 0. It intentionally contains structure, contracts, configuration placeholders, and implementation boundaries only. Business features, agent prompts, and production logic are deferred to sprint work.

Start with [docs/PROJECT_BOOTSTRAP_FOUNDATION.md](docs/PROJECT_BOOTSTRAP_FOUNDATION.md).
